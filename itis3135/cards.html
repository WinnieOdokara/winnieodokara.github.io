Cyber Threat Intelligence Database

Phishing

Severity: Medium

Deceptive communications to steal sensitive data.

Ransomware

Severity: Critical

Malicious software that encrypts user data until a ransom is paid.

SQL Injection

Severity: High

Code injection technique to exploit database vulnerabilities.

Social Engineering

Severity: High

Psychological manipulation of people into performing actions or divulging confidential information.

DDoS Attack

Severity: High

Overwhelming a network or server with a flood of traffic to render it inaccessible.

Zero-Day Exploit

Severity: Critical

Attacking a software vulnerability that is unknown to the vendor and has no patch available.
