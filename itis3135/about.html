About This Course Site

This page is built in part to demonstrate work for the Charlotte class ITIS3135 - Front-End Web Application Development, taught by Professor D.I. von Briesen.

This course provides students with relevant experience in front-end web application development by building multiple websites using different tools and techniques.

More information about this course can be found at Professor von Briesen's course website.

Tools Used in This Course

  • VS Code
  • Emmet within VS Code
  • SFTP through FileZilla or WinSCP
  • Git through GitHub Desktop and GitHub.com
  • Browser developer tools
  • The Accumulus Validator

Sites and Projects Built in This Course

  • A personal home page or portfolio
  • This ITIS3135 course site
  • A fictitious design firm site
  • A purposely bad webpage used to learn what to avoid
  • A whimsical product company website
  • A student, colleague, or peer management tool
  • A client project website
